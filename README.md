# Strummer Academy

Een gitaarschool platform voor lesroosters, video's, huiswerk en berichten.

## Features

- **Dashboard** - Overzicht van lessen en voortgang
- **Rooster** - Lesmomenten beheren per datum
- **Video's** - Lesmateriaal delen met studenten (YouTube/Vimeo)
- **Huiswerk** - Opdrachten toewijzen en bijhouden
- **Berichten** - Communicatie tussen docent en studenten
- **Profiel** - Avatar upload en accountinstellingen

## Rollen

- **Docent** - Volledige toegang tot alle features
- **Student** - Eigen rooster, video's en huiswerk

## Technologie

- Next.js 16
- Supabase (Auth + Database)
- Tailwind CSS
- Lucide Icons

## Setup

1. Supabase project aanmaken
2. Schema migreren uit `supabase/schema.sql`
3. Environment variables invullen in `.env.local`
4. `npm run dev` starten
